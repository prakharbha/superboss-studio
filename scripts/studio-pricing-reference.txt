INSTRUCTIONS FOR UPDATING SANITY CMS
================================================================================
1. Go to http://your-domain.com/studio (or localhost:3000/studio)
2. Navigate to "Studio" in the content types
3. For each studio listed above:
   a. Find the studio by name
   b. Update the following fields:
      - pricePerHour: [1 hour price]
      - price2Hours: [2 hours price]
      - price4Hours: [4 hours price]
      - price6Hours: [6 hours price]
      - price8Hours: [8 hours / Full day price]
   c. Save the changes

Note: price8Hours is REQUIRED and will be used for both "8 Hours" and "Full Day" bookings.
The other hour packages (2, 4, 6) are optional - if not set, the system will calculate
them as pricePerHour Ã— number of hours.
